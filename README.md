# pppChess
## Introduction
This is a chess game built completely in C++ with the **primary purpose of furthering my knowledge in Object Oriented Programming and the C++ language**, hence the name pppChess. I only focused on making the main functions work, so expect the game to look not nice and unpolished.<br><br>
This is my first ever project, or rather the first project that comes a bit close to the state of completion. I had AI assist with the initial structure of the project (when I was only familiar with writing code in a single file) and some part that would otherwise take too much time (like the code implementation of the bot engine).<br><br>
I have implemented local mode, bot mode and a very simple local multiplayer mode (that needs lots of improvements at the mean time). Also, the UI is currently a mess, I will try to improve and add new features if I have time and will...
## Build and run
To build, you must have cmake installed. Run the following command to build the project:
```
cd pppChess
cmake -B build
```
Then run:
```
./build/bin/pppChess
```
Or, you could just go the the release page and download the pre-built binary for a glimpse of the game.
